Luna Launcher 是一个自定义的 Minecraft 启动器，让你可以轻松管理多个 Minecraft 安装。

## 项目简介

Luna Launcher 是 **Prism Launcher** 的增强分支，在 Prism 的基础上增加了更多功能，同时保持与 Prism Launcher 实例和功能的完全兼容。

### 核心特性

- **实例隔离**：每个实例都有独立的 mods、资源包与设置
- **整包管理**：轻松安装与启动 Modpack
- **P2P 联机**：内置 Terracotta 和 YukariConnect 支持
- **镜像加速**：内置 BMCLAPI 支持，国内下载更快
- **Fluent 主题**：现代化的 Fluent Design 主题
- **服务端管理**：直接管理 Minecraft 服务器（开发中）
