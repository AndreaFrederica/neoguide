Luna Launcher 支持管理多种游戏资源，包括资源包、Schematic 文件等。

## 资源包管理

### 安装资源包

1. 打开实例的"资源包"页面
2. 点击"添加资源包"
3. 选择 .zip 文件
4. 调整加载顺序

### 管理资源包

- 启用/禁用资源包
- 拖拽调整顺序
- 删除不需要的资源包

### 资源包格式

- `.zip`：标准资源包格式
- `.mrpack`：Minecraft 官方资源包格式

## Schematic 文件

### 什么是 Schematic？

Schematic 文件记录建筑结构，可以在游戏中粘贴。

### 导入 Schematic

1. 将 .schem 文件放入实例的 `schematics` 目录
2. 在游戏中使用 WorldEdit 加载
