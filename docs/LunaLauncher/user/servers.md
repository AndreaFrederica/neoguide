服务器管理功能让你可以保存常用的 Minecraft 服务器地址，快速加入游戏。

## 添加服务器

### 手动添加

1. 进入"服务器管理"页面
2. 点击"添加服务器"
3. 填写服务器信息：
   - 名称：服务器显示名称
   - 地址：服务器 IP 或域名
   - 端口：默认 25565
4. 保存

### 直接连接

在多人游戏界面直接输入服务器地址也可以快速连接。

## 编辑服务器

### 修改信息

1. 在服务器列表中找到目标
2. 点击编辑按钮
3. 修改服务器信息
4. 保存

### 切换版本

确保客户端版本与服务器兼容。

## 服务器列表

### 快速连接

在服务器列表中点击即可快速加入。

### 服务器状态

- 在线玩家数
- 服务器延迟
- 服务器图标

## 常见问题

### 连接超时？

- 检查服务器地址是否正确
- 确认网络连接正常
- 尝试使用 VPN

### 版本不匹配？

- 安装匹配的 Minecraft 版本
- 安装对应的 Forge/Fabric 版本
