Modpack（整合包）将多个 Mod、配置和资源打包在一起，让玩家可以一键安装整个游戏环境。

## 安装 Modpack

### 从文件安装

1. 点击"添加实例" → "安装 Modpack"
2. 选择本地 .zip 或 .mrpack 文件
3. 选择安装到的实例
4. 等待导入完成

### 从 URL 安装

1. 点击"添加实例" → "安装 Modpack"
2. 输入 Modpack 下载 URL
3. 确认安装

## 导出 Modpack

1. 右键点击实例
2. 选择"导出 Modpack"
3. 选择导出格式（zip/mrpack）
4. 配置导出选项
5. 保存文件

## 更新 Modpack

### 手动更新

1. 重新导入新的 Modpack 文件
2. 选择覆盖或创建新实例

### 自动更新

部分 Modpack 支持自动更新功能。

## Mod 管理

### 安装 Mod

1. 打开实例的"Mods"页面
2. 拖放 .jar 文件到列表
3. 或点击"添加"选择文件

### 管理 Mod

- 启用/禁用 Mod
- 调整加载顺序
- 删除不需要的 Mod

---

使用 Modpack 可以快速体验不同的游戏内容，推荐从知名整合包开始。
