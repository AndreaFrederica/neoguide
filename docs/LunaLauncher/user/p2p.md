Luna Launcher 内置支持 P2P 联机功能，让你无需端口映射或复杂的网络配置即可与朋友一起游戏。

## 支持的 P2P 方案

### Terracotta

[Terracotta](https://github.com/burningtnt/Terracotta) 是一个 Minecraft P2P 联机项目。

**特点：**
- 成熟稳定
- 社区活跃
- 使用 AGPL 许可证

### YukariConnect

[YukariConnect](https://github.com/ElicaseTech/YukariConnect) 是一个替代的 P2P 联机实现。

**特点：**
- 作为 Terracotta 的替代方案
- 可作为独立组件提供
- 旨在提供更灵活的许可

## 使用方法

### 准备工作

1. 确保所有参与者都使用 Luna Launcher
2. 确保使用相同的 Minecraft 版本和 Mod

### 安装联机后端

**首次使用需要先安装联机后端：**

1. 打开 Luna Launcher
2. 进入"设置"
3. 找到"联机"或"P2P"相关选项
4. 选择要安装的后端：
   - **Terracotta** - 点击"安装 Terracotta"
   - **YukariConnect** - 点击"安装 YukariConnect"
5. 等待安装完成

### 开始联机

安装完成后，使用方法非常简单：

#### 创建房间（主机）

1. 选择要联机的实例
2. 点击主界面上的联机按钮：
   - **"Terracotta Online"** - 使用 Terracotta 后端
   - **"YukariConnect Online"** - 使用 YukariConnect 后端
3. 系统自动创建房间并生成邀请信息
4. 分享房间代码或邀请链接给朋友

#### 加入房间（客户端）

1. 打开 Luna Launcher
2. 点击主界面上对应的联机按钮
3. 输入房间代码或点击邀请链接
4. 等待连接建立
5. 启动游戏

## 注意事项

### 网络要求

- 所有参与者需要稳定的网络连接
- 建议使用有线网络以获得最佳体验
- 移动网络可能不稳定

### 性能考虑

- 主机需要更好的性能来运行服务器
- 客户端数量越多，主机负载越高
- 建议不超过 4-5 名玩家

### Mod 兼容性

- 确保 Mod 版本一致
- 某些 Mod 可能不支持联机
- 服务端 Mod 需要所有玩家安装

## 故障排除

### 无法连接到房间

**可能原因：**
- 房间代码输入错误
- 主机已关闭房间
- 网络防火墙阻止连接

**解决方法：**
- 检查房间代码是否正确
- 确认主机仍在运行
- 检查防火墙设置

### 游戏内掉线

**可能原因：**
- 网络不稳定
- 主机性能不足
- Mod 冲突

**解决方法：**
- 检查网络连接
- 主机降低游戏画质
- 检查 Mod 日志
