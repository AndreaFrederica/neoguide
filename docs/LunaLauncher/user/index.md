## 新手入门

### 关于 Luna Launcher

Luna Launcher 是一个专注于"实例隔离"和"使用便捷"的 Minecraft 启动器。

**核心设计理念：**
- 每个实例完全独立，互不干扰
- 简化 Modpack 安装和更新流程
- 提供国内镜像加速（BMCLAPI）
- 支持 P2P 联机，无需复杂网络配置

### 适用场景

- ✅ 需要管理多个 Minecraft 版本
- ✅ 玩 Modpack，需要频繁切换配置
- ✅ 与朋友联机，不想折腾端口映射
- ✅ 国内网络环境，需要下载加速
- ✅ 想要现代化的 UI 体验

## 快速开始

1. **获取启动器**
   - 参考[安装指南](./installation.md)

2. **创建第一个实例**
   - 点击"添加实例"
   - 选择 Minecraft 版本
   - 等待下载完成

3. **安装 Mod（可选）**
   - 打开实例编辑
   - 进入"Mods"选项卡
   - 拖放 Mod 文件或使用 Mod 平台

4. **开始游戏**
   - 选择实例
   - 点击"启动"
