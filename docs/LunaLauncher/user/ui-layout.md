Luna Launcher 提供了灵活的界面布局选项，适应不同用户的使用习惯。

## 布局模式

### 三列布局（默认）

```
┌─────────────────┬─────────────────┬─────────────────┐
│   实例列表      │   实例详情      │   Mod 管理      │
│                 │                 │                 │
│                 │                 │                 │
└─────────────────┴─────────────────┴─────────────────┘
```

### 两列布局

```
┌─────────────────┬───────────────────────────────────┐
│   实例列表      │   实例详情 / Mod 管理              │
│                 │                                   │
└─────────────────┴───────────────────────────────────┘
```

### 单列布局

```
┌─────────────────────────────────────────────────────┐
│                   实例列表                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 自定义选项

### 实例列表

- 显示图标
- 显示最后游玩时间
- 排序方式

### 实例详情

- 显示游戏版本
- 显示 Mod 数量
- 显示实例大小

### 主题颜色

- 浅色模式
- 深色模式
- 跟随系统

## 快捷操作

### 拖拽排序

拖动实例调整显示顺序。

### 搜索筛选

输入关键词快速找到目标实例。
