## 安装自定义模型

### 方法一：手动安装

1. 下载自定义模型文件（.zip）
2. 解压到实例的 `resourcepacks` 目录
3. 在游戏内启用资源包

### 方法二：使用启动器安装

1. 打开实例的"资源包"页面
2. 点击"添加"
3. 选择模型文件
4. 调整加载顺序
