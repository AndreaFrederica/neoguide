实例是 Luna Launcher 的核心概念，每个 Minecraft 游戏环境都是独立的实例。

## 创建实例

### 方法一：新建实例

1. 点击"添加实例"按钮
2. 选择 Minecraft 版本
3. 设置实例名称
4. 点击"确定"等待下载

### 方法二：导入已有游戏

1. 点击"添加实例"
2. 选择"导入"
3. 选择已有的 .minecraft 目录或实例文件夹
4. 确认导入

### 方法三：安装 Modpack

1. 点击"添加实例"
2. 选择"安装 Modpack"
3. 选择本地文件或输入 URL
4. 等待导入完成

## 实例设置

### 基本信息

| 设置项 | 说明 |
|--------|------|
| 名称 | 实例显示名称 |
| 游戏目录 | 游戏文件位置 |
| Java | 使用的 Java 版本 |

### 游戏版本

- 选择 Minecraft 核心版本
- 配置 Fabric/Forge/Quilt 版本
- 设置游戏类型（生存/创造等）

## 管理实例

### 文件操作

- **打开文件夹**：查看实例文件
- **重命名**：修改实例名称
- **复制**：创建实例副本
- **删除**：删除实例（可选择是否保留文件）

### 备份实例

建议定期备份重要实例：
1. 右键点击实例
2. 选择"备份"
3. 选择保存位置
