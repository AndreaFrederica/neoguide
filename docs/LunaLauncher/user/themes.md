Luna Launcher 提供多种主题选择和界面布局选项。

## 内置主题

### Fluent 主题

Luna Launcher 独有的 Fluent Design 主题。

#### Fluent Dark

现代化的深色 Fluent 主题。

**特点：**
- 符合 Fluent Design 规范
- 亚克力材质效果
- 圆角设计
- 适合夜间使用

#### Fluent Light

清新的浅色 Fluent 主题。

**特点：**
- 明亮简洁
- 高对比度
- 适合日间使用

### 切换主题

1. 打开设置
2. 进入"外观"选项卡
3. 选择主题：
   - Fluent Dark
   - Fluent Light
   - 系统默认
4. 应用更改

## 界面布局

### 标准布局

传统的两列布局：
- 左侧：实例列表
- 右侧：实例详情

### 三列布局（实验性）

额外的第三列用于显示更多信息。

**启用方法：**
1. 打开设置 → 界面
2. 勾选"使用三列布局"
3. 重启启动器

**布局结构：**
- 第一列：实例列表
- 第二列：实例信息和操作
- 第三列：新闻、日志等扩展信息

## 自定义选项

### 字体设置

1. 打开设置 → 外观
2. 调整字体大小
3. 选择字体家族

### 图标主题

支持自定义图标主题。

1. 打开设置 → 外观
2. 点击"浏览"选择图标主题
3. 应用更改

### 窗口设置

- 窗口大小
- 启动时最大化
- 记住窗口位置
