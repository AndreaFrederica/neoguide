img2pic 网页版是一个基于 Quasar 框架的 Vue 3 应用，提供友好的图形界面来进行像素艺术转换。

## 功能概述

网页版提供四种处理模式：

- **能量图模式**：基于能量图的高级网格检测，适合复杂图像
- **边缘检测模式**：使用 Sobel 边缘检测的快速处理
- **直接采样模式**：简单直接的网格采样
- **纯放大模式**：不进行像素化，仅放大图像

## 使用方法

### 1. 上传图像

点击上传区域或拖拽图像文件到页面中。支持常见图像格式（PNG、JPG、WEBP 等）。

### 2. 选择处理模式

在左侧面板选择合适的处理模式。

### 3. 调整参数

根据选择的模式，调整相应的参数：

#### 能量图模式参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 最小能量 (min_energy) | 能量阈值，低于此值的区域将被忽略 | 0.15 |
| 平滑窗口 (smooth_window) | 平滑处理窗口大小（奇数） | 9 |
| 间隙容忍 (gap_tolerance) | 网格间隙容差 | 2 |
| 峰值最小间距 (peak_min_dist) | 峰值间最小距离 | 2 |
| 采样模式 | center/average/weighted | center |
| 权重比率 (weight_ratio) | 加权采样权重 | 0.6 |

#### 边缘检测模式参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 边缘阈值 (edge_threshold) | 边缘检测阈值 | 0.1 |
| 最小网格尺寸 | 最小网格大小 | 4 |
| 最大网格尺寸 | 最大网格大小 | 16 |
| 采样模式 | center/average/weighted | center |

#### 直接采样模式参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 网格尺寸 (grid_size) | 网格大小 | 8 |
| 采样模式 | center/average/weighted | center |

#### 纯放大模式参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 放大倍数 | 输出图像放大倍数 | 4 |

### 4. 颜色量化（可选）

启用颜色量化可以减少颜色数量，获得更经典的像素艺术效果：

- **启用量化**：开启/关闭颜色量化
- **颜色数量**：目标颜色数量（2-256）
- **量化模式**：
  - `smart`：智能颜色合并
  - `force`：强制精确颜色数
- **相似度阈值**：颜色合并相似度（0.0-1.0）

### 5. 处理图像

点击"处理图像"按钮开始处理。处理过程在浏览器中完成，图像不会上传到服务器。

### 6. 预览和下载

处理完成后，可以：
- 查看预览图
- 滚动缩放查看细节
- 点击"下载"保存结果

## 快捷操作

- **重置参数**：恢复默认参数值
- **随机示例**：加载示例图像进行测试

## 技术特性

- **纯前端处理**：所有处理在浏览器中完成，保护隐私
- **实时预览**：即时查看处理效果
- **响应式设计**：支持桌面和移动设备
- **离线可用**：支持 PWA 离线使用

## 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 启用 JavaScript
