## 功能亮点

### 多编辑器支持

ANH Lite 集成了多种专业编辑器：

- **Monaco Editor**：VS Code 的核心编辑器，支持多种编程语言的语法高亮
- **Milkdown Editor**：所见即所得的 Markdown 编辑器
- **图片查看器**：支持 JPG、PNG、GIF、WebP 等格式
- **PDF 查看器**：在线预览 PDF 文件

### 跨平台支持

- **Web 版本**：支持所有现代浏览器
- **移动端**：通过 Capacitor 支持 iOS 和 Android
- **桌面端**：通过 Electron 支持 Windows、macOS 和 Linux

### 智能内存管理

- **标签页 GC**：自动卸载长时间未使用的标签页，释放内存
- **大文件优化**：大文件自动禁用语法高亮等特性，提升性能
- **工作区持久化**：自动保存工作区状态，关闭后可以恢复

### 移动端优化

- **虚拟键盘适配**：自动调整布局，确保光标可见
- **移动端选区手柄**：支持触摸调整选区范围
- **触摸优化**：优化的触摸操作界面
