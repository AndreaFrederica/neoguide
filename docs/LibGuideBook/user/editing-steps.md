本文档介绍如何在 LibGuideBook 中添加和编辑步骤。

## 添加新步骤

### 方法一：通过编辑器

1. 打开要编辑的手册
2. 点击"添加步骤"按钮
3. 填写步骤标题和内容
4. 保存

### 方法二：拖拽排序

1. 在步骤列表中拖拽步骤
2. 释放到目标位置
3. 自动保存新顺序

## 编辑步骤内容

### 支持的格式

- **Markdown**：支持标题、列表、链接等
- **工具嵌入**：可添加交互式工具
- **待办事项**：添加可勾选的任务

### 示例

```markdown
## 步骤标题

这是步骤的详细内容。

### 子任务
- [ ] 任务一
- [ ] 任务二

### 嵌入工具
[计时器工具]
```

## 步骤设置

### 基本设置

| 设置项 | 说明 |
|--------|------|
| 标题 | 步骤名称 |
| 内容 | 步骤详细说明 |

### 高级设置

- **条件显示**：根据上一个步骤的完成情况决定是否显示
- **自动完成**：设置完成后自动标记为完成
