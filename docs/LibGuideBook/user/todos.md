子任务（待办事项）帮助你将复杂步骤分解为更小的可执行任务。

## 添加子任务

### 在步骤中添加

1. 编辑某个步骤
2. 在内容中使用以下格式：
   ```
   - [ ] 子任务内容
   - [x] 已完成的子任务
   ```

### 通过界面添加

部分步骤模板可能提供专门的子任务输入框。

## 管理子任务

### 勾选完成

点击子任务前面的复选框标记为完成或未完成。

### 编辑内容

点击子任务文本即可编辑。

### 删除子任务

点击子任务右侧的删除按钮。

## 使用场景

### 适用情况

- 步骤包含多个独立任务
- 需要追踪完成进度
- 需要分配给不同人员执行

### 示例

```markdown
## 实验准备

- [ ] 检查设备状态
- [ ] 准备试剂
- [ ] 记录环境参数
```

通过子任务功能，你可以将大步骤分解为可管理的小任务，提高执行效率。
