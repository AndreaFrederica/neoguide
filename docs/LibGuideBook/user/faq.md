本页面收集了使用 LibGuideBook 时的常见问题。

## 使用问题

### Q: 数据会丢失吗？

A: LibGuideBook 使用浏览器本地存储保存数据。建议定期导出重要手册作为备份。

### Q: 可以在手机上使用吗？

A: LibGuideBook 采用响应式设计，支持在移动设备上使用。

### Q: 如何分享手册？

A: 目前支持导出为 JSON 文件分享。未来版本将支持直接分享链接。

### Q: 手册有数量限制吗？

A: 本地存储空间足够存储大量手册，具体数量取决于浏览器限制。

## 技术问题

### Q: 加载很慢怎么办？

A: 尝试：
1. 清除浏览器缓存
2. 使用 Chrome 或 Edge 浏览器
3. 检查网络连接

### Q: 工具无法使用？

A: 请确保：
1. 浏览器已启用 JavaScript
2. 没有使用广告拦截插件阻止脚本
