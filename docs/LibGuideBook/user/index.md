## 新手入门

### 什么是 LibGuideBook？

LibGuideBook 是一个让你创建**可执行操作手册**的工具。与普通文档不同，LibGuideBook 的手册可以：

- 将复杂流程分解为步骤
- 在每个步骤中嵌入交互式工具
- 支持子任务和待办事项
- 保存运行状态，随时继续

### 适用场景

| 场景 | 说明 |
|------|------|
| **教程制作** | 创建带计算器的编程教程 |
| **实验指导** | 记录科学实验步骤和计时 |
| **工作流程** | 标准化操作流程指导 |
| **游戏攻略** | 带计算工具的游戏指南 |

## 快速开始

### 访问应用

打开浏览器访问：[https://guides.sirrus.cc](https://guides.sirrus.cc)

### 创建第一个手册

1. 进入"手册列表"页面
2. 点击"新建手册"
3. 填写名称和描述
4. 添加步骤和内容
5. 保存

### 运行手册

1. 在手册列表中选择手册
2. 点击"运行"
3. 按照步骤执行操作
4. 使用嵌入的工具辅助
