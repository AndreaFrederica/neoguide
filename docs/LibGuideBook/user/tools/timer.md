计时器工具提供秒表和倒计时功能。

## 模式

### 秒表模式

从 0 开始计时，记录耗时。

**配置：**
- 模式：`stopwatch`

### 倒计时模式

从指定时间开始倒数。

**配置：**
- 模式：`countdown`
- 秒数：倒计时时长（秒）

## 高级配置

### 来源公式工具ID

可以从公式工具获取初始值：

1. 配置公式工具输出变量名
2. 在计时器中设置"来源公式工具ID"
3. 计时器将使用公式结果作为初始秒数

### 引用名称

设置引用名称（如 `@timer1`），可以在公式工具中引用当前值。

## 使用场景

- 烹饪计时
- 实验计时
- 运动计时
- 会议计时
- 任何需要计时的场景
