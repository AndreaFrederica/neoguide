LibGuideBook 支持三种内置交互工具，可以在步骤或子任务中绑定使用。

## 工具类型

| 工具 | 说明 | 适用场景 |
|------|------|----------|
| **计时器** | 秒表或倒计时 | 烹饪计时、实验计时 |
| **计算器** | 基础四则运算 | 简单计算 |
| **公式工具** | 自定义表达式 | 复杂计算、公式应用 |

## 绑定工具

### 在步骤中添加

1. 编辑步骤
2. 选择工具类型
3. 点击"添加"
4. 配置工具参数

### 在子任务中添加

1. 编辑子任务
2. 选择工具类型
3. 点击"添加工具"或快速添加按钮

## 工具配置

每种工具都有特定的配置选项，详见各工具文档。
