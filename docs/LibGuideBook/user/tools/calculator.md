计算器工具提供基础的四则运算功能。

## 功能

- 加法
- 减法
- 乘法
- 除法
- 清空
- 历史记录

## 使用方法

1. 点击数字按钮输入数值
2. 点击运算符按钮进行计算
3. 点击 "=" 查看结果
4. 点击 "C" 清空

## 引用计算结果

设置引用名称后，可以在公式工具中引用：

```
引用名称: @calc1
```

在公式工具中：
```
表达式: @calc1 * 2
```

## 使用场景

- 简单计算
- 临时计算
- 作为公式的输入
