## 新建手册

1. 进入"手册列表"页面
2. 点击"新建手册"按钮
3. 填写基本信息：
   - **名称**：手册标题
   - **描述**：手册说明（支持 HTML）
4. 保存

## 编辑手册

### 基本信息编辑

1. 在手册列表中点击"编辑"
2. 修改名称或描述
3. 点击"保存"

### 添加步骤

1. 进入编辑页面
2. 滚动到底部
3. 点击"添加步骤"
4. 填写步骤信息：
   - **标题**：步骤名称
   - **内容**：步骤说明（支持 HTML）
   - **并行组ID**：可选，用于并行执行

### 编辑步骤

每个步骤可以包含：

- **文本内容**：使用富文本编辑器或 Monaco 编辑器编写
- **工具绑定**：添加计算器、计时器、公式工具
- **子任务**：创建待办事项列表

### 删除步骤

1. 找到要删除的步骤
2. 点击步骤右侧的"删除步骤"按钮
3. 确认删除

## 手册结构

```
手册 (Manual)
├── 步骤 1 (Step)
│   ├── 标题
│   ├── 内容
│   ├── 工具绑定
│   └── 子任务 (StepTodo[])
│       ├── 子任务 1
│       │   ├── 标题
│       │   ├── 描述
│       │   ├── 完成状态
│       │   └── 工具绑定
│       └── 子任务 2
├── 步骤 2 (Step)
└── ...
```

## 标签系统

手册支持标签分类：

- `preset` - 预设手册（不可删除）
- 自定义标签

## 保存和发布

### 保存

编辑过程中自动保存到本地存储。

### 导出

1. 进入手册列表
2. 点击"导出全部"或单个手册的"导出"
3. 下载 JSON 文件
