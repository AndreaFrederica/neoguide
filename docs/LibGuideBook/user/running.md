## 启动运行

1. 在手册列表中选择要运行的手册
2. 点击"运行"按钮
3. 进入运行模式

## 运行界面

### 当前步骤

- 显示当前步骤的标题和内容
- 显示绑定的工具（如果有）
- 显示子任务列表（如果有）

### 导航控制

- **下一步**：进入下一个步骤
- **上一步**：返回上一个步骤
- **重置**：重新开始

### 工具使用

步骤中绑定的工具会自动显示：

- **计时器**：秒表或倒计时
- **计算器**：基础计算功能
- **公式工具**：自定义表达式计算

## 子任务

### 勾选完成

1. 在子任务列表中找到待完成的任务
2. 点击复选框标记为完成
3. 系统自动保存状态

### 任务工具

子任务也可以绑定工具：

- 计时器：记录任务耗时
- 计算器：辅助计算
- 公式：自动化计算

## 运行状态

系统会保存以下运行状态：

- 当前步骤位置
- 工具状态（计时器值、计算结果）
- 子任务完成状态
- 确认状态

### 状态恢复

下次打开同一手册时，会自动恢复到上次的位置。

## 并行步骤

如果步骤设置了 `groupId`，相同 ID 的步骤可以并行执行：

```
步骤 A (groupId: group1) ─┐
                         ├─→ 可以同时进行
步骤 B (groupId: group1) ─┘
```

## 完成手册

完成所有步骤后：

1. 查看总结页面（如果有）
2. 重置或关闭
3. 状态会被保存，可以随时继续
