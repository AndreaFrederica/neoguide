# 导入导出

> **⚠️ 免责声明**：本文档由 **GLM4.7** 自动生成，内容可能存在错误或不完整之处。**请以人工书写的官方为准**。

---

LibGuideBook 支持将手册导出为 JSON 文件，方便备份和分享。

## 导出手册

### 步骤

1. 在手册列表中找到要导出的手册
2. 点击手册卡片上的"导出"按钮
3. 选择保存位置
4. 确认导出

### 导出格式

导出的 JSON 文件包含完整的手册信息：

```json
{
  "id": "manual-uuid",
  "title": "手册标题",
  "description": "手册描述",
  "steps": [...],
  "createdAt": "2024-01-01T00:00:00.000Z",
  "updatedAt": "2024-01-01T00:00:00.000Z"
}
```

## 导入手册

### 步骤

1. 进入手册列表页面
2. 点击"导入"按钮
3. 选择要导入的 JSON 文件
4. 确认导入

### 注意事项

- 导入时不会覆盖同名手册
- 导入的手册会使用新的 ID
- 确保 JSON 格式正确

通过导入导出功能，你可以轻松备份重要手册或与他人分享。
