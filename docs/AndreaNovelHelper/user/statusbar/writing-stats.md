写作统计状态栏显示**今日写作数据**，帮助你跟踪写作进度。

## 位置

写作统计显示在 VS Code 底部状态栏：

```
┌─────────────────────────────────────────────────┐
│ main  master  ✗  ⚙  🔌                         │
│ 📝 今日: 2,345 字 | ⚡ 156 字/分                 │
└─────────────────────────────────────────────────┘
                          ↑
                    写作统计位置
```

## 显示内容

### 默认显示

```
📝 今日: 2,345 字
```

显示今天的总写作字数。

### 详细信息

```
📝 今日: 2,345 字 | 本周: 15,678 字 | ⚡ 156 字/分
```

显示更多信息：
- 今日字数
- 本周字数
- 写作速度

## 统计规则

### 什么时候计入统计？

- 输入新字符时
- 删除字符时（负增长）
- 粘贴文本时

### 什么不计入统计？

- 纯浏览（不编辑）
- 移动光标
- 选择文本

### 写作速度

实时计算的写作速度：
- CPM: Characters Per Minute（每分钟字符数）
- CPH: Characters Per Hour（每小时字符数）

## 数据保存

写作数据会自动保存到本地数据库：
- 自动保存
- 下次打开继续统计
- 不会丢失历史数据

## 查看详细统计

### 打开统计面板

1. 点击状态栏的写作统计
2. 或使用命令面板：`Andrea Novel Helper: 显示写作统计`
3. 会显示详细的统计信息

## 重置统计

### 重置今日统计

1. 使用命令面板（`Ctrl+Shift+P`）
2. 输入 `Andrea Novel Helper: 重置今日统计`
3. 确认重置

### 清除所有数据

⚠️ **警告**：此操作不可撤销！

1. 使用命令面板
2. 输入 `Andrea Novel Helper: 清除写作统计`
3. 确认清除

## 常见问题

### 统计不准确？

1. 确认在有效编辑文件
2. 检查是否有其他窗口编辑同一文件
3. 尝试重置统计

### 数据会丢失吗？

不会。数据会自动保存到数据库，下次打开继续统计。
