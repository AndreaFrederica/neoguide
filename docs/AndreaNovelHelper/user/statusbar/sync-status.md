同步状态显示 **WebDAV 和 Git 的同步状态**，让你随时了解备份情况。

## 位置

同步状态显示在 VS Code 底部状态栏：

```
┌─────────────────────────────────────────────────┐
│ main  master  ✗  ⚙  🔌                         │
│ ☁️ WebDAV: ✓ 2分钟前 | 📦 Git: ✓ 就绪            │
└─────────────────────────────────────────────────┘
                          ↑
                    同步状态位置
```

## 显示内容

### WebDAV 状态

```
☁️ WebDAV: ✓ 2分钟前
```

- `✓` - 同步成功
- `⏳` - 同步中
- `✗` - 同步失败
- `⚠️` - 有警告

### Git 状态

```
📦 Git: ✓ 就绪
```

- `✓ 就绪` - 工作区干净
- `📝 有修改` - 有未提交的修改
- `⚠️ 冲突` - 有合并冲突

## 点击状态栏

点击同步状态可以：
- 查看详细同步信息
- 查看修改的文件列表
- 查看错误详情
- 快速打开对应面板

## 颜色指示

| 颜色 | 状态 |
|------|------|
| 绿色 | 正常/最新 |
| 黄色 | 有警告/需注意 |
| 红色 | 错误/失败 |
| 蓝色 | 同步中 |

## 自动刷新

状态会自动刷新：
- 同步完成后
- 文件保存时
- 定期检查

## 常见问题

### 显示"同步失败"？

1. 检查网络连接
2. 查看错误详情
3. 尝试手动同步

### 时间显示不准确？

1. 确认系统时间正确
2. 同步时间可能延迟
3. 手动同步会更新时间
