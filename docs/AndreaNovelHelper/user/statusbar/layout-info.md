版式信息状态栏显示**当前段落的排版信息**，帮助你控制段落长度和格式。

## 功能概述

版式信息会显示：

- 📏 **段落字数** - 当前段落的字符数
- 📝 **行数** - 当前段落的行数
- 🎯 **格式状态** - 格式化建议

## 界面位置

版式信息显示在 VS Code 底部状态栏：

```
┌─────────────────────────────────────────────────┐
│ main  master  ✗  ⚙  🔌                         │
│ 📏 段落: 156 字 | 4 行                           │
└─────────────────────────────────────────────────┘
                          ↑
                    版式信息位置
```

## 显示内容

### 默认显示

```
📏 段落: 156 字
```

显示当前光标所在段落的字数。

### 详细显示

```
📏 段落: 156 字 | 4 行 | 建议: 适当
```

显示更多信息：
- 字数
- 行数
- 格式建议

### 警告显示

```
📏 段落: 856 字 | ⚠️ 建议分段
```

当段落过长时会显示警告。

## 段落分析

### 段落长度判断

| 字数 | 建议 | 颜色 |
|------|------|------|
| 0-50 | 过短 | 灰色 |
| 50-300 | 适当 | 绿色 |
| 300-500 | 较长 | 黄色 |
| 500+ | 建议 | 红色 |

### 自动检测

扩展会自动检测：
- 光标所在段落
- 实时更新信息
- 提供分段建议

## 格式建议

### 建议分段

当段落过长时：
1. 状态栏显示警告
2. 可以在适当位置分段
3. 保持段落长度适中

### 建议合并

当段落过短时：
1. 状态栏显示提示
2. 可以考虑与下一段合并
3. 保持节奏流畅

## 快速操作

### 快速分段

1. 将光标放在要分段的位置
2. 按 `Enter` 键
3. 会自动处理段落格式

### 快速合并

1. 删除段落间的空行
2. 段落会自动合并
3. 版式信息会更新

## 与智能回车配合

版式信息与智能回车功能配合使用：
- 智能回车自动处理换行
- 版式信息实时反馈
- 保持良好的排版

## 显示模式切换

### 切换简洁模式

1. 右键点击状态栏的版式信息
2. 选择**简洁模式**

只显示字数。

### 切换详细模式

1. 右键点击状态栏的版式信息
2. 选择**详细模式**

显示更多信息。

### 关闭显示

1. 右键点击状态栏的版式信息
2. 选择**隐藏**

## 段落格式规范

### 推荐段落长度

- **对话**：50-150 字
- **叙述**：100-300 字
- **描写**：200-400 字

### 段内空格

- 中文段落首行不缩进（使用空行分隔）
- 英文段落首行缩进（使用 Tab 或空格）

## 高亮段落

### 高亮当前段落

启用后，当前段落会有背景色高亮：
- 方便识别段落范围
- 帮助判断段落长度

### 启用/禁用

在 VS Code 设置中：
1. 搜索 `AndreaNovelHelper.layout.highlightCurrentParagraph`
2. 设置为 `true` 启用

## 常见问题

### 字数统计不准？

1. 确认光标在段落内
2. 检查是否正确识别段落边界
3. 段落由空行分隔

### 如何隐藏版式信息？

1. 右键点击状态栏的版式信息
2. 选择**隐藏**
3. 或在设置中关闭

### 分段建议不符合需求？

1. 可以在设置中调整分段阈值
2. 或关闭自动建议
3. 根据自己的写作习惯调整

## 相关设置

在 VS Code 设置中搜索 `AndreaNovelHelper.layout`：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| `showParagraphInfo` | 显示段落信息 | `true` |
| `paragraphWarningLength` | 段落警告长度 | `500` |
| `paragraphShortLength` | 段落过短长度 | `50` |
| `highlightCurrentParagraph` | 高亮当前段落 | `false` |
