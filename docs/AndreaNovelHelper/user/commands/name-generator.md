## 使用方法

### 命令：生成随机名字

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Andrea Novel Helper: 生成随机名字`
3. 选择文化背景
4. 选择性别
5. 选择风格
6. 输入生成数量
7. 选择是否包含姓氏
8. 选择要使用的名字

### 命令：快速生成名字

使用默认设置快速生成 5 个名字：

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Andrea Novel Helper: 快速生成名字`
3. 选择要使用的名字

### 命令：生成随机角色

完整的角色生成流程：

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Andrea Novel Helper: 生成随机角色`
3. 按提示逐步生成角色信息

### 命令：分步生成角色

逐步生成角色，每一步都可以自定义：

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Andrea Novel Helper: 分步生成角色`
3. 按提示逐步完成角色创建

### 命令：显示名字生成统计

查看名字生成的统计信息：

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 `Andrea Novel Helper: 显示名字生成统计`
3. 查看统计信息

## 选择名字

生成名字后，会显示名字选择面板：

```
┌─────────────────────────┐
│ 选择要使用的名字        │
├─────────────────────────┤
│ ☐ 张三                 │
│ ☐ 李四                 │
│ ☐ 王五                 │
│                         │
│ [确定]  [全选]  [取消]  │
```

### 操作

- **单选**：点击名字前的复选框
- **全选**：点击"全选"按钮
- **确定**：将选中的名字复制到剪贴板

## 常见问题

### 生成的名字不合适？

1. 尝试不同的文化背景
2. 调整性别选项
3. 更换风格

### 如何批量生成？

在数量输入框中输入需要的数量（建议 1-100）。

### 名字可以保存吗？

选择名字后会自动复制到剪贴板，可以粘贴到文档中。

### 支持自定义名字规则吗？

目前不支持自定义规则，但可以提供反馈。
