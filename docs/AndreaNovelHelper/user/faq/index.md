本页面收集了用户在使用 Andrea Novel Helper 时常见的问题和解决方案。

## 安装问题

### Q: 扩展无法安装？

A: 请确保你使用的是最新版本的 VS Code，并且网络连接正常。可以尝试从 VS Code 扩展市场手动安装。

### Q: 安装后功能不生效？

A: 尝试以下步骤：
1. 重新加载 VS Code 窗口（Ctrl+Shift+P → "Reload Window"）
2. 检查扩展是否在设置中启用
3. 查看扩展输出面板获取详细错误信息

## 使用问题

### Q: 文件追踪不工作？

A: 请确保：
1. 文件格式为 `.md` 或 `.txt`
2. 文件不在 `novel-helper` 排除目录中
3. 扩展已获得必要的文件权限

### Q: 字数统计不准确？

A: 字数统计基于实际字符数，可能与 Word 等软件的统计方式不同。可以尝试切换统计模式或检查大文件近似设置。

### Q: 预览面板打不开？

A: 尝试：
1. 检查是否有其他扩展冲突
2. 重启 VS Code
3. 查看输出面板的错误信息

## 同步问题

### Q: WebDAV 同步失败？

A: 请检查：
1. WebDAV 服务器地址是否正确
2. 用户名和密码是否有效
3. 网络连接是否正常

### Q: Git 自动提交不工作？

A: 请确认：
1. 项目已初始化 Git 仓库
2. 已配置自动提交规则
3. 没有未解决的 Git 冲突

## 其他问题

### Q: 如何导出反馈？

A: 可以通过命令面板执行 "Andrea Novel Helper: 导出调试信息" 来生成反馈报告。

### Q: 性能问题？

A: 如果遇到性能问题，可以尝试：
1. 禁用不需要的功能
2. 减小追踪的文件数量
3. 清理旧的历史数据
