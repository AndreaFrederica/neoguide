Typst 预览提供实时的 Typst 文档预览功能。

## 功能说明

Typst 预览集成在 VS Code 中，提供：
- 实时预览
- 自动更新
- 同步滚动
- 错误提示

## 打开方式

### 自动预览

导出 Typst 文档时，会自动映射到内存盘：
- 可以使用 VSCode Typst 插件预览
- 自动更新预览
- 无需手动刷新

### 手动预览

如果已安装 VSCode Typst 插件：
1. 打开 Typst 虚拟文件
2. 插件自动显示预览

## 预览功能

### 实时渲染

- 编辑时自动更新
- 编译后立即显示
- 支持所有 Typst 功能

### 同步滚动

- 编辑器和预览同步
- 点击跳转到位置
- 双向同步

### 错误显示

- 编译错误高亮
- 错误信息提示
- 快速定位问题

## 配置

### Typst 插件

安装 VSCode Typst 插件：
1. 打开扩展市场
2. 搜索 "Typst"
3. 安装官方插件

### 预览设置

在 Typst 插件设置中：
- 选择预览模式
- 配置自动刷新
- 设置字体路径

## 常见问题

### 预览不更新？

1. 检查是否导出了新版本
2. 确认虚拟文件已更新
3. 刷新预览面板

### 编译错误？

1. 查看错误信息
2. 检查 Typst 语法
3. 验证模板格式

### 中文显示问题？

1. 配置中文字体
2. 检查字体路径
3. 使用系统字体
